Detalles del Proyecto

Proyecto backend desarrollado con Node.js, Express y pruebas realizadas desde Postman. Permite registrar, modificar, eliminar y consultar datos simulando un sistema de usuarios.

Se trabajó con middlewares de autenticación, estructura modularizada de rutas y controladores, y validaciones robustas para inputs del cliente.

Es ideal para demostrar manejo de REST API y control de flujos HTTP sin necesidad de un frontend visual. Fue usado como práctica de arquitectura de servidores.
