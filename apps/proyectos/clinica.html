Clínica | Sistema de Turnos

Proyecto web desarrollado con Firebase y HTML para turnos médicos. Incluye gestión de usuarios, login, registro, y panel de administración.

Ir al proyecto en vivo

Detalles del Proyecto

Esta aplicación permite a los pacientes registrarse y solicitar turnos según especialidades médicas disponibles. El panel de administración permite agregar horarios, médicos y controlar la disponibilidad. El sistema incluye autenticación con Firebase y validación en tiempo real.

También se incorpora una vista diferenciada para usuarios administrativos, que pueden ver métricas de actividad, estados de turnos y exportar la información.

Este sistema fue utilizado como base para clínicas pequeñas que buscan digitalizar el proceso de asignación de turnos de forma rápida y segura.
